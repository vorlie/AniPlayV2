# AniPlayV2 Next Release Notes

## AniList UX

- Added a watch preflight panel to AniList detail pages.
- AniList playback now shows the selected provider, playback match, and episode before opening the player.
- Episode selection prefers local watch history, then AniList progress + 1, then episode 1.
- Added clearer match recovery controls: Play, Change match, Forget match, and Search again.
- Redesigned playback match selection with provider labels, episode counts, confidence, match reasons, manual retry search, and empty states.
- Provider-aware AniList mappings are preserved so AllAnime and Anikoto matches do not override each other.
- Added an experimental Settings toggle for AniList-first Anikoto search results.
- Search results now show cover thumbnails when AniList or provider metadata includes artwork, falling back to numbered tiles otherwise.
- Added a persisted search result view switch with Compact and Poster Grid modes.
