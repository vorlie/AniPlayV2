# AniPlayV2 v1.9.8 Release Notes

## Provider Updates

- Anikoto is now the default provider for fresh searches and AniList dashboard playback.
- AllAnime remains selectable, but it is no longer the fresh-install default.
- AniList playback mappings now remember their catalog provider, so old AllAnime mappings are not reused for Anikoto playback.


**Full changelog:** https://github.com/vorlie/AniPlayV2/compare/1.9.6...1.9.7
