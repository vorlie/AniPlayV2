# AniPlayV2 v1.10.3 Notes

## Notifications

- AniPlay now shows in-app notifications for available updates and updates that are ready to install.
- Added **Settings -> Advanced** notification sound controls with Off, Important, and All modes.
- Added notification sound presets: Soft chime, Crystal, and Arcade, plus a Test sound button.

## Polish

- Added a few small hidden seasonal/session touches for users to discover. These are cosmetic only and do not change playback, provider behavior, logs, AniList progress, downloads, or Discord Rich Presence.
